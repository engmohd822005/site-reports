إعداد قاعدة البيانات السحابية

أنشئ مشروع Firebase مجاني ثم الصق بيانات الإعداد أدناه. يكفي مرة واحدة فقط، وتُحفظ تلقائياً.

خطوات سريعة:
1. افتح console.firebase.google.com
2. أنشئ مشروعاً جديداً
3. اضغط "إضافة تطبيق ويب" ثم انسخ الـ Config
4. فعّل Authentication → Email/Password
5. فعّل Firestore Database
